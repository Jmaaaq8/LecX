HTML ELEMENTS

• Nested HTML: HTML elements can be nested within each other to create complex structures. For example, a < span > element can be nested within a < p > element to style a specific part of the paragraph.
• Elements: HTML elements are the building blocks of a webpage. They consist of a start tag, end tag, and content.
• Never Skip the End Tag: It's crucial to close each element with its corresponding end tag to ensure proper rendering of the page.
• Empty HTML Elements: Some elements, like < br > (line break) and < img > (image), don't have any content. These are called empty elements and are closed with a self-closing tag, like < br /> or < img src="image.jpg" />.
• HTML is Not Case Sensitive: While HTML tags and attributes are typically written in lowercase, they can be written in uppercase without affecting the rendering of the page. However, it's generally recommended to use lowercase for consistency.

HTML Attributes

HTML attributes provide additional information about elements. The href attribute specifies the link destination, src defines the image source, width and height set image dimensions, alt provides alternative text, style applies inline styles, and title adds a tooltip.

• href Attribute: Specifies the URL of a hyperlink.
• src Attribute: Specifies the URL of an image source.
• width and height Attributes: Define the dimensions of an image in pixels.
• alt Attribute: Provides alternative text for an image, which is displayed if the image cannot be loaded or if the user is using a screen reader.
• style Attribute: Applies inline styles to an element, overriding default styles or styles defined in a CSS stylesheet.
• title Attribute: Provides additional information about an element, which is displayed as a tooltip when the user hovers over the element.

HTML Styles

HTML styles are used to enhance the visual appearance of web pages. The style attribute allows you to apply specific styles directly to an element, such as background color, text color, font, font size, and text alignment.

• Style Attribute: The style attribute allows you to apply inline styles directly to an HTML element.
• Background Color: Sets the background color of an element.
• Text Color: Sets the color of the text within an element.
• Fonts: Specifies the font family to be used for the text.
• Text Size: Sets the font size of the text.
• Text Alignment: Determines the alignment of the text within an element (left, right, center, or justified).

HTML Styles

• Inline CSS: Styles are applied directly to an HTML element using the style attribute.
• Internal CSS: Styles are defined within the < head > section of an HTML document using the < /style > tag.
• ID Attribute: Styles are applied to a specific element with a unique ID.
• External CSS: Styles are defined in a separate CSS file and linked to the HTML document using the < link > tag.
• Colors: Define colors using color names or hexadecimal codes.
• Fonts and Sizes: Specify font families and font sizes.
• Border, Padding, and Margin: Control the appearance and spacing of elements.
