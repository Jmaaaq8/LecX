    Basic Link Styling
    < style>
       a {
          color: #3498db; /* Changes the link color */
          font-family: Arial, sans-serif; /* Sets font */
          text-decoration: none; /* Removes underline */
       }
    
       a:hover {
          color: #2980b9; /* Changes color on hover */
       }
    < /style>
    
    < a href="https://www.example.com">Styled Link< /a>
    This styles all links on the page with a custom color, font, and a hover effect.
    


    Removing Underline and Adding Background Color
    < style>
       a {
          text-decoration: none;
          color: white;
          background-color: #e74c3c; /* Background color for link */
          padding: 8px 12px; /* Adds padding for better visibility */
          border-radius: 4px; /* Rounds corners */
       }
    
       a:hover {
          background-color: #c0392b; /* Darker background on hover */
       }
    < /style>
    
    < a href="https://www.example.com">Styled Background Link
    This example removes the underline, changes text color, and adds a background color with rounded corners.
    


    Creating a Link Button
    < style>
       .link-button {
          display: inline-block;
          color: white;
          background-color: #2ecc71;
          padding: 10px 20px;
          border: none;
          border-radius: 5px;
          text-align: center;
          text-decoration: none;
          font-weight: bold;
       }
    
       .link-button:hover {
          background-color: #27ae60; /* Darker green on hover */
       }
    < /style>
    
    < a href="https://www.example.com" class="link-button">Styled Link Button< /a>
    By using the link-button class, this link appears as a button with padding, color, and rounded corners, resembling a clickable button.
