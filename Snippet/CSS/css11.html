    CSS
    /* Grid Container for Gallery */
    .gallery {
      display: grid;
      grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)); /* Responsive columns */
      gap: 10px; /* Space between images */
      padding: 10px;
    }
    
    /* Gallery Items */
    .gallery-item {
      overflow: hidden;
      border-radius: 8px;
    }
    
    .gallery-item img {
      width: 100%;
      height: auto;
      display: block;
    }


    Explanation
    Grid Container (.gallery):
    
    display: grid: Sets the container as a grid layout.
    grid-template-columns: repeat(auto-fill, minmax(200px, 1fr)): Creates a flexible column structure where each column is at least 200px wide, expanding as necessary to fit the container. This makes the layout responsive, adjusting the number of columns based on screen width.
    gap: 10px: Adds consistent spacing between images.
    Grid Items (.gallery-item):
    
    Each item holds an image, styled with rounded corners and set to fully occupy its grid cell without stretching.
