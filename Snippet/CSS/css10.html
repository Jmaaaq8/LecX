HTML
    < div class="grid-container">
      < div class="grid-item">Header< /div>
      < div class="grid-item">Sidebar< /div>
      < div class="grid-item">Main Content< /div>
      < div class="grid-item">Footer< /div>
    < /div>


    CSS
    /* Grid Container */
    .grid-container {
      display: grid;
      grid-template-columns: 100px 200px 1fr; /* Three columns with specified widths */
      grid-template-rows: 150px auto 50px; /* Three rows with specified heights */
      gap: 10px; /* Space between rows and columns */
      padding: 10px;
      background-color: #f8f8f8;
    }
    
    /* Grid Items */
    .grid-item {
      background-color: #3498db;
      color: white;
      font-size: 1.2em;
      text-align: center;
      padding: 20px;
      border-radius: 5px;
    }
    
    /* Specific grid areas */
    .grid-item:nth-child(1) { grid-column: 1 / -1; } /* Header spans across all columns */
    .grid-item:nth-child(2) { grid-row: 2; grid-column: 1; } /* Sidebar in first column, second row */
    .grid-item:nth-child(3) { grid-row: 2; grid-column: 2 / 4; } /* Main Content spans columns 2 and 3 in second row */
    .grid-item:nth-child(4) { grid-column: 1 / -1; grid-row: 3; } /* Footer spans across all columns in third row */

    
    Explanation
    Grid Lines: The grid is divided into lines that separate each row and column, allowing placement of items within specific grid sections.
    
    Grid-Template-Columns:
    
    grid-template-columns: 100px 200px 1fr: Creates three columns:
    The first column is 100px wide.
    The second column is 200px wide.
    The third column (1fr) takes up the remaining available space.
    Grid-Template-Rows:
    
    grid-template-rows: 150px auto 50px: Creates three rows:
    The first row is 150px high.
    The second row (auto) adjusts its height based on content.
    The third row is 50px high.
    Grid Item Placement:
    
    .grid-item:nth-child(1): The header spans across all columns (using grid-column: 1 / -1).
    .grid-item:nth-child(2): The sidebar is placed in the first column, second row.
    .grid-item:nth-child(3): The main content spans columns 2 and 3 in the second row.
    .grid-item:nth-child(4): The footer spans across all columns in the third row.
