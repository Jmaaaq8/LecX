HTML
    < div class="grid-container">
      < div class="grid-item">1< /div>
      < div class="grid-item">2< /div>
      < div class="grid-item">3< /div>
      < div class="grid-item">4< /div>
      < div class="grid-item">5< /div>
      < div class="grid-item">6< /div>
    < /div>


    CSS
    /* Grid Container */
    .grid-container {
      display: grid; /* Sets the container as a grid layout */
      grid-template-columns: repeat(3, 1fr); /* Creates 3 equal columns */
      grid-template-rows: auto; /* Row height will adjust automatically */
      gap: 10px; /* Adds a gap between rows and columns */
      padding: 10px;
      background-color: #f0f0f0;
    }
    
    /* Grid Items */
    .grid-item {
      background-color: #3498db;
      color: white;
      font-size: 1.5em;
      text-align: center;
      padding: 20px;
      border-radius: 5px;
    }



    Explanation
    Grid Container:
    
    display: grid: Activates CSS Grid on the container.
    grid-template-columns: repeat(3, 1fr): Defines a three-column layout, where each column is 1 fraction (1fr) of the available space, making columns equal in width.
    grid-template-rows: auto: Sets rows to automatically adjust in height based on content.
    gap: 10px: Adds a 10px gap between both rows and columns.
    Grid Items:
    
    .grid-item styles each item with background color, text color, padding, and rounded corners to make the grid visually appealing.
