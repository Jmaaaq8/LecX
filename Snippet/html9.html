    Basic Hyperlink
    < a href="https://www.example.com">Visit Example
    This link will take the user to example.com when clicked.
    

    Opening Link in a New Tab
    < a href="https://www.example.com" target="_blank">Open Example in New Tab
    The target="_blank" attribute ensures the link opens in a new browser tab.
    

    Image as a Link
    < a href="https://www.example.com">
       < img src="image.jpg" alt="Example Image" />
    < /a>
    This example makes the image image.jpg clickable, taking the user to example.com.
    

    Button as a Link
    < button onclick="window.location.href='https://www.example.com';">Go to Example< /button>
    Here, the button will act like a link, directing the user to example.com when clicked.

    
    Button with  Tag Inside
    < button>
       < a href="https://www.example.com">Go to Example
    < /button>
    In this version, an  tag is inside the button. Note that the link styling will need to be adjusted with CSS to make the < a > tag behave like a button.
