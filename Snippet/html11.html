    Basic Image Syntax
    < img src="https://www.example.com/image.jpg" alt="Description of the image">
    This is the simplest way to add an image. If the image cannot be loaded, the alt text "Description of the image" will appear.
    


    Specifying Image Dimensions
    < img src="https://www.example.com/image.jpg" alt="Description of the image" width="300" height="200">
    The width and height attributes set the image dimensions to 300 pixels wide and 200 pixels high.
    


    Using CSS to Set Image Dimensions
    < style>
       .resized-image {
          width: 150px;
          height: auto; /* Maintains aspect ratio */
       }
    < /style>
    
    < img src="https://www.example.com/image.jpg" alt="Description of the image" class="resized-image">
    Here, CSS controls the image width, and setting height: auto keeps the image aspect ratio.
    


    Floating an Image
    < style>
       .float-image {
          float: left; /* Positions image to the left */
          margin-right: 15px; /* Adds space between image and text */
       }
    < /style>
    
    < img src="https://www.example.com/image.jpg" alt="Description of the image" class="float-image">
    < p>This text will wrap around the image, aligning to its right side due to the float property. You can also float the image to the right if needed.< /p>
    In this example, the float: left CSS property moves the image to the left, allowing text to wrap around it on the right. The margin-right adds spacing.
    


    Centering an Image
    < style>
       .centered-image {
          display: block;
          margin-left: auto;
          margin-right: auto;
          width: 50%; /* Adjusts image width to half the container */
       }
    < /style>
    
    < img src="https://www.example.com/image.jpg" alt="Centered image example" class="centered-image">
    Here, the image is centered by setting margin-left and margin-right to auto, and the width is adjusted to 50% of its container.
